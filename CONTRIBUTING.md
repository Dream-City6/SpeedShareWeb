# Contributing

Thank you for helping improve SpeedShareWeb.

## Before opening an issue

- Search existing issues.
- Test with the latest published version.
- Remove passwords, private filenames, local IP addresses and personal information from screenshots and logs.
- For security problems, follow SECURITY.md instead of creating a public issue.

## Bug reports

Include:

- App version
- Android version and device model
- Browser and operating system used to access the web interface
- Network type, such as home Wi-Fi or phone hotspot
- Clear reproduction steps
- Expected and actual behavior
- Screenshots or logs with personal information removed

## Feature requests

Describe the problem first, then explain the proposed solution. Mention whether the feature affects the Android interface, browser interface, server behavior, security or compatibility.

## Pull requests

- Keep changes focused.
- Explain the reason for the change.
- Test the Android app and browser interface.
- Update documentation when behavior changes.
- Do not commit signing keys, credentials, generated APKs or private test files.

## Translations

Translations should be natural and consistent rather than literal. Current target languages are English, Simplified Chinese and Japanese.
