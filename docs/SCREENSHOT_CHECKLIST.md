# Screenshot checklist

Prepare at least five screenshots:

1. Android start screen
2. Server running with the address visible
3. Browser file list on desktop
4. Browser grid view on mobile
5. Upload or file-management interaction

Before capturing:

- Use sample filenames only
- Hide private IP addresses when possible
- Clear notifications
- Remove personal photos and documents
- Use consistent language
- Use the same app version
- Keep status-bar time and battery presentation consistent
- Avoid showing passwords
