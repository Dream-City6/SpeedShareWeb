# License decision

Do not publish a LICENSE file until all direct dependencies and copied code have been reviewed.

## Recommended options

### GPL-3.0

Choose this when you want distributed modified versions to remain open source under the same license.

### Apache-2.0

Choose this when you want broad reuse with an explicit patent grant and do not require modified versions to remain open source.

## Before deciding

- List all direct dependencies and their licenses
- Identify any copied or adapted source code
- Confirm that bundled web libraries, fonts, icons and images permit redistribution
- Confirm whether the app contains generated or third-party assets
- Add copyright notices required by dependencies

Provisional recommendation for Speed Share Web: GPL-3.0, subject to the dependency review.
