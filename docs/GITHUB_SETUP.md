## Pre-public checklist

- No signing key or certificate is tracked
- No signing password is present in Gradle files
- No local.properties file is tracked
- No API keys, tokens or private server addresses are tracked
- No personal test files are included
- The app builds from a fresh clone
- Screenshots contain no private filenames, IP addresses or notifications
- README descriptions match the real implementation
- Privacy and security documents match the real dependencies
- A compatible open-source license has been selected
- A signed APK has been tested on more than one Android device
