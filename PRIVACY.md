# Privacy Policy

_Last updated: 2026-06-23_

## Summary

SpeedShareWeb is designed to provide file access and transfer over a local network. It does not require an account or cloud storage.

## Data processed by the app

Depending on the features enabled by the user, the app may process:

- Names, sizes, types and locations of files selected or exposed by the user
- Files uploaded to or downloaded from the Android device
- Local-network connection information required to operate the file server
- Temporary operational logs used to display status or diagnose errors

## Data transmission

Files and related information are transmitted between the Android device and the browser used to access the local server.

The project does not intentionally send file contents to a developer-operated cloud service. This statement must be reviewed again before every public release, especially after adding analytics, crash reporting, update checks or other online services.

## Storage

Files remain in locations selected or authorized by the user. Temporary data may be created while uploads or downloads are in progress.

## Network security

The local server uses plain HTTP and does not provide password or token authentication. HTTP traffic is not end-to-end encrypted, and other devices on the same local network may be able to access the server or observe or modify traffic.

Users should:

- Use the app only on trusted private local networks
- Avoid exposing the server through router port forwarding
- Stop the server after use
- Avoid transferring sensitive files over public Wi-Fi

## Permissions

Android permissions are used only for functions presented by the app, such as accessing user-selected files and operating the local server.

The exact permission list will be documented before the first public release.

## Analytics and advertising

The initial public release is intended to contain no advertising and no developer-operated behavioral analytics. This section must be updated if that changes.

## Children

SpeedShareWeb is a general-purpose utility and is not designed to collect personal information from children.

## Changes

This policy may be updated when features or dependencies change. Material changes will be documented in the release notes.

## Contact

Use the repository's private security-reporting method for privacy or security concerns. A public contact address will be added before release.
